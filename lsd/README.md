# lsd (LSDeluxe)

[lsd](https://github.com/lsd-rs/lsd) is a rewrite of `ls` with lots of added features. Currently replaced by `eza` – see `../eza/`.

## Installation

```zsh
sudo port install lsd
```

## Configuration

Configuration via `~/.config/lsd/config.yml`. See [config options](https://github.com/lsd-rs/lsd?tab=readme-ov-file#config-file-content).

No config file tracked here.
