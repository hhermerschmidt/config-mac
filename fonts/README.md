# Fonts

Nerd Fonts and other fonts used in terminal and editor configurations.

## Installed Fonts

- **Meslo Nerd Font** – used by Powerlevel10k. Source: [romkatv/powerlevel10k](https://github.com/romkatv/powerlevel10k?tab=readme-ov-file#meslo-nerd-font-patched-for-powerlevel10k)
- **Fragment Mono** – Source: [Google Fonts](https://fonts.google.com/specimen/Fragment+Mono)
- **FiraCode Nerd Font Mono** – used in Zed and kitty. Source: [nerdfonts.com](https://www.nerdfonts.com/font-downloads)

## Installation

Double-click each `.ttf` file to open Font Book and install.
