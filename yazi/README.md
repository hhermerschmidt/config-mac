# yazi

[Yazi](https://github.com/sxyazi/yazi) is a blazing fast terminal file manager.

## Installation

```zsh
sudo port install yazi
```

## Configuration

`keymap.toml` lives in `~/.config/yazi/keymap.toml`.

### Installation

```zsh
just install yazi
```

### Manual

```zsh
mkdir -p ~/.config/yazi
cp .config/yazi/keymap.toml ~/.config/yazi/keymap.toml
```

## Keymaps

| Key | Action |
|---|---|
| `S` | Open shell in current directory (Ranger style) |
