# broot

[Broot](https://github.com/Canop/broot) is a new way to see and navigate directory trees.

## Installation

```zsh
sudo port install broot
```

## Configuration

No config file tracked here. Initialise `br` shell function in `$ZSH_CUSTOM/init_tools.zsh`:

```zsh
source ~/.config/broot/launcher/bash/br
```
